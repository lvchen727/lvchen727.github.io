A dipyrimidine sequence library for determining the sequence dependence of cyclobutane pyrimidine dimer formation

The cis-syn CPD is the major photoproduct resulting from UV irradiation that results in C to T mutations found in human skin cancers. In this project, I studied CPD distribution within all possilbe NPyPyN sites. The quantification data given by the project provided some general rules for estimating CPD distribution in duplex DNA.

  • Wrote python scripts to generate sequence libraries containing all possible NPyPyN (N is A,T,C,G and PyPy represents CC, TC, CT, TT) sites.
  • Sythesized, cloned and PCR amplified one of designed 129-mer sequence which contains all 64 NPyPyN sites.
  • Utilized T4-endonuclease and gel electrophoresis to map the CPD distribution within 129-mer under different direct and indirect excitation conditions.
  • Analyzed and plotted the final result using python libraries.
Skin cancer, biochemistry, Molecular biology, Data analysis

K-mer Kernels for Genetic Distance

Phylogenetic tree reconstruction requires construction of a multiple sequence alignment (MSA) from sequences. However, alignment can be quite challenging for very large genomes. An interesting recent approach uses intensive hashing method to count the fraction of all unique k-mers appearing in either genome that are common to both to estimate the evolutionary distance, which avoids the need of assembly and alignment. In this project, I tried to implement this k-mer idea using a more space efficient method based on the 2BWT.

  • Wrote python scripts and Java program to process raw data, build 2BWT, count distinct and shared k-mers and generate the genetic distance table.
  • Constructed the similar common k-mer table and phylogeny tree as in AAF paper using the simulated reads data provided from paper.
  • Applied and tested the method on 4 real genome large data set, and the resulting phylogeny tree similar to the one generated by the program phyloT.
Algorithm design, BWT, Computational biology, Phylogeny tree
